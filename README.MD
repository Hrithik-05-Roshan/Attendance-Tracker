# Smart Attendance System

This is a **QR code based attendance system** built with:

- Node.js + Express
- MongoDB (Mongoose)
- HTML CSS JAVASCRIPT

## Features
- Student/Teacher/Admin login
- QR Code generation & scanning
- Automatic attendance marking
- Dashboard for students and teachers
